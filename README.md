# Nastha_test

Nashta Demo App adalah aplikasi untuk mencatat event yang akan datang.

Stack yang digunakan adalah 

Frontend: 
- React.js

Backend: 
- Express.js
- Cors
- Pg
- Sequelize

untuk menjalankan :
- client : npm start
- server : nodemon app.js
